Bienvenido a este repositorio de **Machine Learning aplicado al mundo económico**. El objetivo principal de este espacio es proporcionar modelos prácticos, código funcional y recursos de aprendizaje para entender cómo la inteligencia artificial y el análisis de datos pueden resolver problemas en el ámbito de la economía y las finanzas.

---

## 🎯 Objetivos del Repositorio
* **Aprender haciendo:** Cuadernos interactivos con explicaciones paso a paso.
* **Aplicación real:** Casos de uso enfocados directamente en la economía.
* **Recursos continuos:** Actualización constante con nuevo material de estudio.

---

## 📂 Estructura del Repositorio

El contenido está distribuido en cuadernos de Jupyter (`.ipynb`), organizados por las distintas técnicas de Machine Learning:

| # | Técnica / Tema | Archivo | Descripción |
|---|----------------|---------|-------------|
| 1 | **Modelos de Regresión** | [`MLconModelosdeRegresion_BM.ipynb`](./MLconModelosdeRegresion_BM.ipynb) | Implementación de modelos de regresión para predicciones económicas. |
